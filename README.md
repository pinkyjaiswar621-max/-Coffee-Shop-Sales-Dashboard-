# Data-Analysis (Interactive dashboard creation using MS Excel)
## Project Object
This project analyzes coffee shop sales performance across multiple dimensions by tracking key metrics such as total transactions, quantity sold, revenue, and profit. It helps identify top-performing product categories and cities while revealing important monthly, yearly, and weekday sales trends. The insights are presented through an interactive and visually appealing Excel dashboard designed to support effective business analysis and data-driven decision-making.
## Dataset used
- <a href="https://drive.google.com/uc?export=do...">Dataset</a>
## Questions (KPIs)
- What is the total number of transactions recorded during the selected period?
- How much total quantity of products has been sold?
- What is the total revenue generated by the coffee shop?
- What is the total profit earned during the selected time frame?
- Which product category contributes the highest number of transactions?
- Which product category generates the maximum revenue?
- Which city is the top-performing location in terms of revenue?
- How does revenue vary across different months?
- Which weekday records the highest number of transactions?
- How does revenue and profit compare on a monthly basis?
- Dashboard Interaction <a href="https://github.com/pinkyjaiswar621-max/-Coffee-Shop-Sales-Dashboard-/blob/main/cofee%20shop%20sales%20dashboard.png">View Dashboard</a>
## Process
- Verify data or any missing values and anomalies and sort out the same.
- Make sure data is consistent and clean with respect to the data type, data format and values used.
- Created pivot tables according to the question asked 
- Merge all pivot tables into one dashboard and apply slicer to make dynamic
